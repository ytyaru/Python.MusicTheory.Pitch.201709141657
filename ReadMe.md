# このソフトウェアについて

音程から半音数を取得する。

音度を一部変更。int型も受け付けるようになった。

単体テストコードを書いた。計47項目。

テストコード|項目数
------------|------
TestPitchClass.py|9
TestAccidental.py|9
TestDegree.py|13
TestInterval.py|16

# 実行

```sh
$ cd ./src/
$ python TestInterval.py
$ python TestDegree.py
```
